# Create a GeocodeServer

Create an object of class `GeocodeServer` from a URL. This object stores
the service definition of the geocoding service as a list object.

## Usage

``` r
geocode_server(url, token = arc_token())
```

## Arguments

- url:

  the URL of a geocoding server.

- token:

  an object of class `httr2_token` as generated by
  [`auth_code()`](https://rdrr.io/pkg/arcgisutils/man/auth.html) or
  related function

## Value

an object of class `GeocodeServer`.

## Examples

``` r
server_url <- "https://geocode.arcgis.com/arcgis/rest/services/World/GeocodeServer"
geocode_server(server_url)
#> <GeocodeServer>
#> Description: World Geocoder
#> Version: 11.5
#> CRS: 4326
```
