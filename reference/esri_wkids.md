# Esri well-known IDs

An integer vector containing the WKIDs of Esri authority spatial
references. Esri WKIDs were identified from the
[`{arcgeocoder}`](https://cran.r-project.org/package=arcgeocoder)
package from [@dieghernan](https://github.com/dieghernan).

## Usage

``` r
esri_wkids
```

## Format

An object of class `integer` of length 2886.

## Value

a numeric vector of well-known IDs
